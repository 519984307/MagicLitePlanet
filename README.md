🚀MagicLight Assistant - 适用于小型团体的协作系统

  交流、工作、活动、分析、管理、审批...等的功能均不可思议地集成于优雅而简约轻便的 **Magic<Light\> Assistant**。

- 平台：Windows x64
- 开发环境：Visual Studio 2022 MSVC
- 类型：Qt桌面应用程序
- 数据库：MySQL
- 语言：C++ 11
- 框架：Qt5.14

-> 此项目由一名非科班励志自学成才的大学生呈现——我的第一个可上线项目。
